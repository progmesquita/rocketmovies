# 🎥 RocketMovies

Projeto front-end desenvolvido para criar notas sobre filmes.

## Funcionalidades

- Fazer login/cadastrar usuário
- Editar perfil
- Cadastrar filmes
- Visualizar lista de filmes

## Instalação

Instale as dependências do projeto com npm

```bash
  cd ./nome_do_diretorio
  npm install
```
